Exercici 3.1: Ús enllaçat d'operadors ternaris

Escriu una expressió que utilitzi enllaços d'operadors ternaris per determinar si un número és
positiu, negatiu o zero.

function numberSign(num) {
    return num === 0 ? "zero"
    : num > 0 ? "positiu" : "negatiu";
}

console.log(numberSign(0));

Exercici 3.2: Operador ternari amb funcions

Crea una funció trobarMaxim que accepti tres paràmetres (a, b, c) i utilitzi l'operador ternari
per determinar el valor màxim. 

function trobarMaxim(a, b, c) {
    return a > b && a > c ? `${a} es el mayor`
    : b > a && b > c ? `${b} es el mayor`
    : `${c} es el mayor`
}

console.log(trobarMaxim(8, 64, 128));
