Exercici 5: Funció de fletxa amb 'setTimeout'

Crea una funció de fletxa que imprimeixi un missatge a la consola després d'esperar 3 segons.

const printMessage = () => {
    setTimeout(() => {
        console.log("Mensaje retrasado a los 3 segundos");
    }, 3000);
}

printMessage();
