Array loops (1)

Abre la consola del navegador para probar los siguientes ejercicios:

Exercici 1: forEach

Teniu un array let noms = ['Anna', 'Bernat', 'Clara'];
Utilitza forEach per a imprimir cada nom a la consola: 

let noms = ['Anna', 'Bernat', 'Clara'];

noms.forEach(nom => console.log(nom));

Exercici 2: for-of

Teniu un array let noms = ['Anna', 'Bernat', 'Clara'];
Utilitza for-of per a imprimir cada nom a la consola: 

let noms = ['Anna', 'Bernat', 'Clara'];

for(const nom of noms) {
    console.log(nom);
}

Exercici 3: filter

Teniu un array let numeros = [1, 2, 3, 4, 5, 6].
Utilitza filter per a crear una nova array que només contingui els números parells. 

let numeros = [1, 2, 3, 4, 5, 6]

let numerosPar = numeros.filter((num) => num % 2 === 0);
console.log(numerosPar);
